# Менеджер для установки и настройки панели 3X-UI

## 🔹 Возможности

3X-UI Setup Manager делает следующее:

- Устанавливает Docker
- Устанавливает и запускает 3X-UI в docker-контейнере
- Автоматически получает сертификаты для домена
- Настраивает fallback-страницу

Также, скрипт позволяет:

- Обновить версию 3X-UI до последней
- Запустить, остановить или перезапустить Docker контейнер
- Изменить домен
- Изменить путь до подписок

> [!CAUTION]
> Менеджер не меняет путь до подписок в самой 3X-UI. Вам нужно это делать руками через интерфейсе панели. Этот скрипт нужен лишь для того, чтобы обновить конфигурацию сервера после того, как вы измените путь в интерфейсе. В ином случае вы не сможете получать доступ к подпискам по новому адресу

## 🔹 Необходимые настройки

Перед запуском менеджера убедитесь, что:

- Настроен и защищен доступ к серверу по SSH
- Установлен и настроен firewall (открыты порты 80, 443 и 8443)
- Зарегистрирован и делегирован домен, указывающий на ваш сервер

## 🔹 Запуск

Подключитесь по **SSH** к серверу и выполните команду:

```sh
sh <(wget -O - https://raw.githubusercontent.com/neketabrain/3x-ui-setup-manager/main/3x-ui-setup-manager.sh)
```

или

```sh
wget -O /tmp/3x-ui-setup-manager.sh https://raw.githubusercontent.com/neketabrain/3x-ui-setup-manager/main/3x-ui-setup-manager.sh && sh /tmp/3x-ui-setup-manager.sh
```
